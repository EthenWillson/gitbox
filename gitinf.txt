一、设置用户名和邮箱：
$ git config --global user.name "Ethen Willson"
$ git config --global user.email ""

二、创建git版本库
1.创建一个空目录
$ mkdir gitbox(目录的名字)
$ cd gitbox
$ pwd //用于显示当前目录

2.通过git init命令把该目录变成git可以管理的仓库
$ git init

三、把文件添加到版本库（要放入版本库目录或者子目录下）
1.把文件添加到仓库
$ git add gitinf.txt(文件名)

2.把文件提交到仓库
$ git commit -m "这里写你想要写的信息"

四、修改文件&更新版本库
1.修改文件
2.通过git status命令掌握仓库当前状态
$ git status 

3.看具体修改了什么内容
$ git diff

4.重新提交
$ git add gitinf.txt（文件名）
$ git commit -m ""
